# People-Management

### 개요
 학부 연구실에 들어간 후에 연구실에 속한 학부생과 졸업한 졸업생들의 정보를 쉽게 관리하기 위해 개발하였다. 도구 내에서 학부생과 졸업생들의 정보관리, 
이메일을 보낼 수 있는 기능을 제공하여 편의성을 제공한다.
 
### 주요기능
* 학부생 졸업생 정보관리
  + 파일에서 처리하던 정보관리를 도구를 구축하여 처리한다.
  + 보기 쉬운 UI 와 CRUD 기능제공
* 이메일 발송
  + 도구내에서 이메일 발송을 통해 편의성을 제공한다.
